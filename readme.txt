

Modo de Compilacion: 	
     			Para compilar el archivo se tiene que ingresar en la carpeta de nuestro archivo 
			y luego escribir en consola:
			$make 
			
Modo de Ejecucion (Requiere previa Compilacion mediante make jar) de la Aplicación:
			Escribir en la consola: 
			$make run

Generar Jar de la Aplicación:
			Escribir en la consola:
			$make jar
			
Ejecutar Applet:
			Escribir en la consola:
			$make runApplet
			
Generar javadoc: 
			Escribir en la consola:
			$make doc

Eliminacion de los archivos .class:
			Para eliminar los archivos compilados, utilice la opcion: 				
			$make clean
